IPTV via streamdev

VDR implementation of the VTP (Video Transfer Protocol) Version 0.0.3 and a basic HTTP Streaming Protocol.

System Requirements

File creation must be enabled via the docker environment variable START_XMLTV=true!

Default M3U & XMLTV URLs:

  • M3U: http://myHost:PORT/channels.m3u
  • XMLTV: http://myHost:PORT/epg.xmltv

roboTV

roboTV is a Android TV based frontend for VDR...

System Requirements

The VDR plugin robotv must be enabled via the PLUGINS environment variable.

Example: PLUGINS=robotv epgsearch live streamdev-server

Edit "vdr/config/plugins/robotv/robotv.conf":

  • PiconsURL = http://myHost:PORT/channellogos/picons
  • EpgImageUrl = http://myHost:PORT/epgimages/%d.jpg

For some boxes (e.g. Chromecast with Google TV) it is also necessary to install the app "Live Channels" to view live tv.
